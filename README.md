# recommender-system-supermarket
Sistema de recomendación híbrido (explícito, implícito y social) para e-commerce de supermercado
## Dataset

Este proyecto utiliza **datos sintéticos generados artificialmente**, diseñados para simular el comportamiento de usuarios en un e-commerce de supermercado.

Incluye:
- Interacciones implícitas (views, cart, purchase)
- Ratings explícitos
- Relaciones sociales entre usuarios

El objetivo es replicar un entorno realista para desarrollar y evaluar sistemas de recomendación.
